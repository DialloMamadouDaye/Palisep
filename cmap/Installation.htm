INSTALLATION DE CMAP Carte de France PHP

Page-test | Exemples | Autres versions

  1. Dézippez le fichier France-free.zip à la racine de votre serveur.

  2. Un dossier appelé "cmap" doit apparaître avec à l'intérieur :

    - Un fichier Installation.htm
    - Un fichier france_free.swf
    -     Un fichier map.php

    - un fichier Page-test.php vous permettra de tester le fonctionnement de la carte.

  3. Vous pouvez installer les fichiers france-free.swf et map.php où bon vous semble sur votre serveur
    du moment que vous respectez le chemin relatif vers la carte.

  4. Editez les différentes variables de configuration de la carte selon vos besoins ( voir plus bas )

  5. Copiez - collez ce code dans votre page :

    <?PHP
     include("map.php");
    ?>

    Si vous avez placé le fichier map.php dans un sous repertoire, indiquez le chemin relatif jusqu'à ce repertoire.
    Exemple : include("cartes/france/map.php");

RENVOI DYNAMIQUE: la variable qui permet d'afficher les départements sur la carte s'appelle "dpt";
Votre page doit donc recevoir cette variable une fois la carte cliquée; méthode $_GET.
Voir code source de Page-test.php


CONFIGURATION DE CMAP Carte de france

 


    Pour installer la carte sur une de vos pages, copiez-collez ce script :

    <?PHP

    include("map.php");

    ?>

    Vous pouvez ajouter à ce script différentes variables de configuration selon vos besoin :
    NOTA : Si aucune de ces variables n'est précisée, la carte fonctionnera avec ses propres paramètres par défaut.
    Vous pouvez utiliser ces variables indépendemment, c'est à dire préciser une, deux, trois... ou toutes.

    $SIZE_MAP="600";
    Définit la taille d'affichage de la carte en pixels.
    Par défaut, la taille de la carte est de 450 x 450 pixels.

    $DIR_MAP="";
    Indique l'emplacement ( chemin relatif ) du fichier France-free.swf au cas où vous mettiez celui-ci dans un sous-repertoire.
    Exemple : $DIR_MAP="carte/france/";
    Note : Mettre un slash à la fin.


    $URL_REDIR="Ma-page.php";
    Définit la page de redirection dynamique de la carte et qui reçoit la variable 'dpt=XX' renvoyée par la carte.
    Exemple : Ma-page.php
    Si vous n'avez renseigné aucune page , c'est la page Page-test.php qui prend le relai.


    $URL_TARGET="_self";
    definit le mode d'ouverture des pages : _blank , _self , _parent , _top .


    $CODE="XX";
    Renvoi à la carte la valeur de la variable '$dpt', et colore le département ainsi précisé.
    Exemple : $CODE=$_GET['dpt'];


    $ROLL_BT="#FF0066";
    Définit la couleur hexadécimale des boutons au survol de la souris.


    $PRES_BT="#FF0000";
    Définit la couleur hexadécimale du département actif.


    $BG_COLOR="#FF0099";
    Définit la couleur hexadécimale de la carte.
    Exemple : #CCF099


    $BORDERS="yes";
    Affiche ou non les pays frontaliers en gris clair.
    "yes" ou "no";


     NOTE : ces variables doivent être placé avant l'appel INCLUDE


EXEMPLES

<?PHP

$SIZE_MAP="390";
$URL_REDIR="carte-des-magasins.php";
$ROLL_BT="#FF0066";
$PRES_BT="#FE0933";

include("map.php");
?>
